# RegExr Lite — Live Regex Tester

Test regular expressions with real-time match highlighting and capture group inspection. Free, instant, works offline.

**Live Demo:** [regexr-lite.surge.sh](https://regexr-lite.surge.sh/)

## Features

- **Live matching** — Results update as you type
- **Match highlighting** — Every match is highlighted in the test string
- **Capture groups** — See all groups for each match
- **Flag toggles** — g, i, m, s flags with one click
- **10 common patterns** — Email, URL, IPv4, phone, date, hex color, number, HTML tag, password strength, slug
- **Error detection** — Invalid regex shown immediately with error message
- **Zero dependencies** — Single HTML file, works offline

## Tech

- Pure HTML/CSS/JavaScript
- Native RegExp engine
- Responsive split-pane layout
